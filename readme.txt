ゴースト「芹那と妃音with玄恵」

▼あらすじ
 ある日、あなたはベランダ（またはそれに類する場所）の鍵をかけ忘れて出かけたら部屋を狐耳と尻尾の生えたガラの悪そうなJKたちに部屋を占拠されていた。

▼キャラクター紹介
	
芹那（せりな）
 画面右側の狐っ娘JK。不良グループ「蛇突鬼<ダッキ>」のメンバー。一人称がオレ。処女。
	
妃音（きと）
 画面中央の狐っ娘JK。不良グループ「蛇突鬼<ダッキ>」のリーダー。髪のお下げに見える部分も尻尾。処女。
	
玄恵（くろえ）
 画面左側の狐っ娘JK。しばらく起動させると出てくる。「蛇突鬼」とは関係ない優等生。処女。
